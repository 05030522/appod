# 매일 학습 기록 저장소 📖

이 저장소는 매일 꾸준히 학습한 내용과 코딩 테스트 연습 결과를 기록하기 위해 만들어졌습니다.

## 🚀 진행 방식

-   **ST (Study):** 매일 학습하는 일반적인 내용이나 프로젝트 진행 사항을 기록합니다.
    -   폴더 예시: st_날짜
-   **CT (Coding Test):** 코딩 테스트 문제 풀이 소스 코드와 해설을 기록합니다.
    -   폴더 예시: ct_날짜

## 📝 목표

-   매일 꾸준히 학습하는 습관 형성
-   학습 내용 체계적으로 정리 및 복습
-   코딩 테스트 실력 향상 과정 추적
